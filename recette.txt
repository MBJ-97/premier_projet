1- Creer le dossier en local et l'ouvrir avec VScode.
2- ouvrir un terminal sur VScode. (En haut cliquer sur terminal ensuite
nouveau. Le terminal s'affichera en dessous.)
3- Initier Git sur notre repo (Repository = Dossier) en ecrivant "git
init" puis cliquer sur entree.
4- Creer un repo publique sur Github.
5- A partir de Github dans la page du nouveau repo copier puis coller dans le terminal puis appuiyez entree pour chaqu'une des 3 etapes donnees (sauf la derniere!!)
6- Ajouter les changements effectues => "git add ."
7- Nommer les changement qu'on a ajoute => "git commit -m 'votre message
descriptif' "
8- Push les changements sur github => 'git push -u origin main'

REMARQUE IMPORTANTE : Chaque changement necessite les commandes
suivantes :
=> git add
=> git commit
=> git push
